================================
1. CLASSE WRAPPER: VoitureState
================================
Responsabilité: Tracker l'état d'une voiture pendant l'allocation

Attributs:
  - Voiture voiture
  - LocalDateTime heureLiberee (quand la voiture revient)
  - int placesRestantes (nombre de places non utilisées)
  - List<Reservation> clients (clients dans cette voiture)

Méthodes:
  - estDisponible(LocalDateTime heure): boolean
    → retourne true si heureLiberee.isBefore(heure)
  - retirePlace(int nbPersonnes): void
    → placesRestantes -= nbPersonnes
  - ajoutClient(Reservation res): void
    → clients.add(res)
  - getPlacesRestantes(): int

================================
2. CLASSE SERVICE: VoitureStateService
================================
Responsabilité: Gérer la liste d'états des voitures

Méthodes:
  
  a) creerEtatVoitures(List<Voiture> voitures): List<VoitureState>
     → Crée un VoitureState pour chaque voiture avec:
        - heureLiberee = DateTime très ancien (= libre maintenant)
        - placesRestantes = voiture.getNbPlaces()
        - clients = liste vide
  
  b) trouvVoituresDisponibles(List<VoitureState> etats, LocalDateTime heure): List<VoitureState>
     → Retourne voitures où estDisponible(heure) == true
     → Filtre aussi celles avec places >= 0
  
  c) selectionnerVoiture(List<VoitureState> aptes, int nbPlacesRequises): VoitureState
     → Applique critères de sélection dans l'ordre:
        1) Plus petite capacité (placesRestantes >= nbPlacesRequises)
        2) Déjà roulée (nbKmParcourus max)
        3) Type carburant (à définir)
        4) Random
  
  d) assignerVoiture(VoitureState etat, Reservation res, int tempsAttenteMin): VoitureState
     → Met à jour l'état:
        - etat.ajoutClient(res)
        - etat.retirePlace(res.getNombrePersonnes())
        - etat.heureLiberee = res.getDateHeureArrivee() + tempsAttenteMin
     → Retourne l'état mis à jour
  
  e) afficherEtatVoitures(List<VoitureState> etats): void
     → Affiche chaque voiture avec ses clients, heure libération, places restantes
     → DEBUG: helpful pour vérifier l'algorithme
